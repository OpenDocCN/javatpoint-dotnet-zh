# 如何在微软 Word 中插入分页符

> 原文:[https://www.javatpoint.com/to-insert-page-break-in-ms-word](https://www.javatpoint.com/to-insert-page-break-in-ms-word)

Word 在每页的末尾插入一个分页符。它还允许您在文档中的其他位置插入分页符。插入分页符的步骤如下；

*   将光标放在要插入分隔符的位置
*   选择页面布局选项卡
*   在“页面设置”组中，单击“分隔符”命令
*   将出现分页符列表
*   左键单击从列表中选择所需的分页符
