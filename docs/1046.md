# 如何推进幻灯片

> 原文:[https://www.javatpoint.com/how-to-advance-slides-powerpoint](https://www.javatpoint.com/how-to-advance-slides-powerpoint)

*   选择幻灯片
*   选择动画选项卡
*   在“过渡到此幻灯片”组中，找到“高级幻灯片”部分
*   它显示了两个选项；鼠标点击后自动
*   在“自动完成”选项中输入所选幻灯片的所需时间
*   选择下一张幻灯片并选择所需的时间
*   如果要对所有幻灯片应用相同的计时，请单击“全部应用”
