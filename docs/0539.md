# Excel 中的拼写检查

> 原文:[https://www.javatpoint.com/excel-how-to-use-spell-check](https://www.javatpoint.com/excel-how-to-use-spell-check)

微软 Excel 是最强大的程序之一，帮助我们在电子表格的行和列中记录数据。它允许我们存储大量数据，并在其上执行不同的以数据为中心的操作。

因为我们通常借助键盘输入数据，所以我们有可能打错一些拼写。为此，Excel 为我们提供了一个内置工具，可以轻松找到并纠正错误或输入错误的拼写。该工具通常被称为“拼写检查”和“拼写检查器”。然而，Excel 中的拼写检查工具不如 Word 拼写检查功能强大，因为它不会在我们刚开始打字时就检查语法错误或拼写。在 excel 中，我们必须手动检查和更正拼写。

![Spell Check in Excel](../Images/3ba6221c69f7e7ac66744bc5349f6bd4.png)

在本文中，我们将讨论 Excel 中拼写检查的详细介绍。本文还讨论了帮助我们理解如何在 Excel 工作表中进行拼写检查并提高整体工作效率的步骤，从而提高我们的工作效率。

## 什么是拼写检查？

拼写检查是一个软件工具，帮助我们找到单词的错误拼写并纠正它们。该工具通常有助于各种文字处理程序、电子邮件、数据输入、手机信息、博客文章等。一个好的拼写检查工具会自动在整个文档中找到拼错的单词，并给我们相关的建议来纠正它们。有些工具甚至会自动纠正拼错的单词。

拼写检查工具可以内置于特定的程序中，也可以作为独立软件提供。然而，最常见的是，它们通常都用红色标出错误。此外，还会显示拼写正确的建议菜单，使我们能够选择正确的拼写或忽略它。

在 Excel 中，拼写检查工具位于功能区的“**检查**”选项卡下。它由与上面写有“ABC”的瓷砖相关联的文字“**拼写**来识别。它看起来像下图:

![Spell Check in Excel](../Images/c360934e2d603e11f4eddbf5eeee9752.png)

每次我们点击“查看”选项卡下的拼写检查工具，它都会启动“**拼写检查对话框**，如下图所示:

![Spell Check in Excel](../Images/73b4fd8deb99a9954186492b4803ce76.png)

#### 注意:虽然我们可以用不同的方法在 Excel 中执行拼写检查，但最简单的方法是使用快捷键。Excel 允许通过按键盘上的功能键“F7”立即在整张纸上执行拼写检查。但是，必须注意的是，我们可能首先需要通过按下键盘上的“Fn”按钮来激活功能键。

## 拼写检查是如何工作的？

我们必须了解拼写检查工具在 Excel 中如何工作的过程。假设我们选择单元格 A1，并在整张纸上运行拼写检查。该工具将分析单元格 A1，然后分析第一行中的其他单元格。之后，它将移动到第二行，从左到右检查所有单元格，然后是第三行，依此类推。如果该工具在任何单元格中发现任何拼写错误，它将显示带有各种选项的**拼写检查对话框**。

除此之外，如果我们在工作表的其他地方选择单元格并运行拼写检查工具，该工具将从该特定点开始检查错误。例如，如果我们选择单元格 B2 并运行拼写检查，它将检查 B2 单元格中第二行的所有单元格，然后检查其他行。每当我们从工作表中其他位置的单元格开始时，它都会询问我们是否要在选择后检查剩余单元格后，继续检查工作表开头的拼写错误。

## 理解拼写检查对话框界面

当我们在 Excel 中运行拼写检查时，它会显示一个带有一些选项的拼写检查对话窗口。拼写检查对话窗口通常如下所示:

![Spell Check in Excel](../Images/67d65bd1c64baf4bc2db521e4699dd1c.png)

在上图中，我们有以下选项:

*   **忽略一次**:假设 Excel 找到一个特定的单词并标记为错误，并且该单词在工作表中重复多次。但是，我们希望保持所选单元格的原样，并将光标移动到相应单词的下一个单元格；我们必须单击“忽略一次”按钮。这样，我们可以检查所有出现的错别字，并决定是否使用建议来更改它。
*   **全部忽略**:如果 Excel 发现某个单词并标记为错误，并且该单词在工作表中重复了多次，但我们想保持所有这些出现的情况不变(保持原样)，则必须单击“全部忽略”按钮。
*   **添加到词典**:当 Excel 将一些单词显示为错误，并在不正确的单词下标记它们，即使在我们看来它们是正确的，我们可以使用“添加到词典”按钮将该特定单词保存为正确的单词。这样，Excel 就不会再次标记这些单词，并像其他正确的单词一样接受它们。一般来说，对于我们自己创建和使用的某些名称或缩写，我们使用“添加到词典”按钮。
*   **建议**:每当 Excel 发现拼写不正确或错误时，都会在建议下列出一些相关的单词作为备选。我们可以从列表中选择任何想要的单词。
*   **更改**:当我们从列表中拾取任何建议时，我们需要决定是仅针对所选单元格还是在整个工作表中进行更改。如果我们只想对选定或突出显示的单元格使用建议，我们必须单击“更改”按钮。
*   **全部更改**:当我们在工作表中出现任何错误/拼错的单词，并且我们想一次全部更改时，我们必须在建议下选择正确的单词后点击“全部更改”按钮。这将在整个工作表中应用相应的建议。
*   **自动更正** : Excel 自动更正选项帮助我们在保存一个特定单词一次后自动更正拼写。如果我们从建议列表中选择适当的建议，然后单击“自动更正”按钮，Excel 将在“自动更正”列表中添加相应的单词。这意味着，每当我们再次输入错误的单词时，Excel 会根据我们之前选择的建议自动更正它。
*   **词典语言**:我们可以从词典语言旁边的下拉列表中选择词典语言。

## 调整拼写检查设置

Excel 拼写检查工具有一些默认配置或选项，有助于控制 Excel 如何连接和格式化我们的数据。我们可以通过单击拼写检查对话框中的“选项”按钮来更改这些默认设置。它将打开**选项对话框**，通过导航到校对菜单来帮助我们在拼写检查中进行必要的更改。

![Spell Check in Excel](../Images/5c9fae61e9719b4e4ac7f4e92d377b1d.png)

以下是 Excel 中拼写检查的一些默认显示:

*   Excel 拼写检查会忽略所有大写的单词。这意味着，如果我们拼错了大写的单词，它将不会被检查。例如，单词“HELLOOO”不会被标记为拼写错误的单词。
*   Excel 拼写检查忽略检查包含数字的单词。例如，如果我们键入单词“Helloo123”，它将不会被标记为拼写错误的单词。
*   Excel 拼写检查忽略互联网网址和文件地址。
*   Excel 拼写检查不会忽略重复的单词。比如——你好你好，你好吗？由于单词“Hello”在这里重复出现，因此它(多余的单词)将被标记为错误。

## 在 Excel 中拼写检查选定工作表的步骤

如果我们在工作簿中的多个工作表中有数据，并且我们只想在某些单个工作表中执行拼写检查，我们需要遵循以下步骤:

*   首先，我们需要导航工作表标签并选择要执行拼写检查的工作表。要选择纸张，我们必须按住 **Ctrl** 按钮，并用鼠标单击所需的纸张。
    T3】
*   选择所需的纸张后，我们需要进入**查看>拼写**或按快捷键 **F7** 。这样，Excel 将在所有选定的工作表中进行拼写检查，并向我们显示拼写错误的单词和建议。
    拼写检查完成后，我们需要右键单击工作表标签并选择“取消工作表分组”选项来取消工作表分组。

## 在 Excel 中拼写检查整个工作簿的步骤

Excel 还允许我们同时检查整个工作簿/电子表格中的拼写错误。这将覆盖活动工作簿的所有工作表。为此，我们需要遵循以下步骤:

*   首先，我们需要导航到当前工作簿的工作表标签，并用鼠标右键单击。接下来，我们必须点击选项“**选择所有工作表**”。这样，我们可以选择活动工作簿的所有工作表。
    T3】
*   选择完所有表格后，我们必须导航至**查看>拼写**或按键盘上的快捷键 **F7** 。通过这样做，Excel 将对整个工作簿进行拼写检查，并向我们显示拼写错误的单词和建议。
    我们可以使用拼写检查对话框中给出的选项来执行所需的操作。

这就是我们如何在 Excel 中使用拼写检查工具。
