# 如何删除微软 Word 中的文本

> 原文:[https://www.javatpoint.com/to-delete-text-in-ms-word](https://www.javatpoint.com/to-delete-text-in-ms-word)

您可以轻松删除 Word 中的文本，包括字符、段落或文档的所有内容。Word 为您提供了不同的删除文本的方法；下面给出了一些常用的方法；

*   将光标放在文本旁边，然后按退格键
*   将光标放在文本的左侧，然后按下删除键
*   选择文本并按退格键或删除键
*   选择文本并在上面键入新文本。
