# F#属性

> 原文：<https://www.javatpoint.com/f-sharp-attribute>

在 F#中，属性用于为程序代码构造启用元数据。属性可以应用于任何结构，如函数、模块、方法和类型。

**语法:**

```
[<target:attribute-name(arguments)>]

```
