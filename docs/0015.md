# C#关键词

> 原文：<https://www.javatpoint.com/csharp-keywords>

关键字是保留字。您不能将其用作变量名、常量名等。

在 C#中，关键字不能用作标识符。但是，如果我们想使用关键字作为标识符，我们可以在关键字前面加上@字符。
